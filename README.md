# FarmCart Platform

FarmCart is a mobile-first digital agriculture platform connecting farmers directly to buyers, improving market access, pricing transparency, and income sustainability.

## Features
- Farmer product listings
- Buyer ordering system
- Mobile payments integration
- Order tracking
- Scalable backend architecture
